# CNN Image Dehazing System
An AI-powered web application that removes haze and fog from images using the **AOD-Net** (All-in-One De-hazing Network) architecture.

## 🚀 Features
* **Deep Learning Engine**: Uses a 5-layer CNN to estimate and remove atmospheric light.
* **Streamlit UI**: Simple drag-and-drop interface for hazy images.
* **Camera Support**: Real-time image capture and dehazing.
